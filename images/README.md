# Image Placement Guide

Place your images in the following locations:

## Root (`images/`)
- `logo.png` — Your ElbCar Detailing logo (the one with Hamburg skyline)

## Gallery (`images/gallery/`) — Before & After pairs
These are used for the draggable before/after sliders:

| Filename | Description |
|---|---|
| `engine-before.jpg` | Mercedes engine bay — dirty |
| `engine-after.jpg` | Mercedes engine bay — clean |
| `volvo-interior-before.jpg` | Volvo driver seat — dirty |
| `volvo-interior-after.jpg` | Volvo driver seat — clean |
| `vw-front-before.jpg` | VW Touran front interior — dirty |
| `vw-front-after.jpg` | VW Touran front interior — clean |
| `vw-rear-before.jpg` | VW Touran rear seats — dirty |
| `vw-rear-after.jpg` | VW Touran rear seats — clean |
| `trunk-before.jpg` | Trunk/boot — dirty |
| `trunk-after.jpg` | Trunk/boot — clean |
| `fuelcap-before.jpg` | Fuel cap area — dirty |
| `fuelcap-after.jpg` | Fuel cap area — clean |

## Instagram (`images/insta/`) — Instagram preview images
- `insta1.jpg` — e.g. Audi R8
- `insta2.jpg` — e.g. Ferrari 488
- `insta3.jpg` — e.g. Porsche 911
- `insta4.jpg` — e.g. Lamborghini Urus

**Recommended sizes:**
- Gallery images: 1200x750px (16:10 aspect ratio)
- Instagram images: 600x600px (square)
- Logo: PNG with transparent background
